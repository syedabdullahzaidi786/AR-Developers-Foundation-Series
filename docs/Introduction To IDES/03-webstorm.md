# WebStorm

**WebStorm** is an integrated development environment for JavaScript and related technologies. Like other JetBrains IDEs, it makes your development experience more enjoyable, automating routine work and helping you handle complex tasks with ease.

> **Note**: WebStorm is a paid product with a 30-day free trial. It is free for students and open-source maintainers.

## Key Features

-   **Smart Coding Assistance**: WebStorm understands your code and provides smart completion, on-the-fly error detection, and powerful navigation.
-   **Debugger**: Debug your client-side and Node.js applications with ease.
-   **Tool Integration**: Integrate with linters, build tools, test runners, and REST clients.
-   **Version Control**: A unified UI for Git, GitHub, Mercurial, and other VCS.

## Installation Guide

### Windows

1.  **Download**: Visit the [WebStorm download page](https://www.jetbrains.com/webstorm/download/).
2.  **Run Installer**: Download the `.exe` file and run it.
3.  **Setup Wizard**: Follow the installation wizard instructions.
4.  **Installation Options**: You can check "Add 'Open Folder as Project' to Context Menu" and update the PATH variable.
5.  **Run**: Launch WebStorm after installation.

### macOS

1.  **Download**: Visit the [WebStorm download page](https://www.jetbrains.com/webstorm/download/) and download the `.dmg` file.
2.  **Install**: Open the `.dmg` file and drag the WebStorm app to your `Applications` folder.
3.  **Launch**: Open WebStorm from your Applications.

### Linux

1.  **Download**: Download the `.tar.gz` file from the [WebStorm website](https://www.jetbrains.com/webstorm/download/).
2.  **Extract**: Extract the tarball to a directory of your choice.
    ```bash
    tar -xzf WebStorm-*.tar.gz
    ```
3.  **Run**: Navigate to the `bin` subdirectory and run `webstorm.sh`.
    ```bash
    cd bin
    ./webstorm.sh
    ```
